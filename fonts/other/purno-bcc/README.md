The CSS and web font files to easily self-host the "Purno BCC" font. Please visit the main [Hixbe website](https://hixbe.com/fonts/purno-bcc) to view more details on this package.

## Quick Installation

Hixbe offers multiple methods to import the CSS, including using a bundler like Vite or using SASS. You can find full documentation [here](https://hixbe.com/docs/getting-started/introduction).

```javascript
npm install @hixbe/purno-bcc
```

Within your app entry file or site component, import it in.

```javascript
import "@hixbe/purno-bcc"; // Defaults to weight 400
import "@hixbe/purno-bcc/400.css"; // Specify weight
```

Supported variables:
- Weights: `[400]`
- Styles: `[normal]`
- Subsets: `[bengali]`

> Note: `italic` may not be supported by all fonts. To learn more about what weights and styles are supported, please visit the [Hixbe website](https://hixbe.com/fonts/purno-bcc).

Finally, you can reference the font name in a CSS stylesheet, CSS Module, or CSS-in-JS.

```css
body {
  font-family: "Purno BCC";
}
```
