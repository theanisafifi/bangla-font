# Hixbe Bangla Font

This repository hosts all the font files for the Hixbe Bangla Font project. For more information, see the [Hixbe README.md](https://github.com/hixbe/font-files).

## Contributing

This repository only accepts font submissions.

If you would like to contribute to this repository, please see the [Contributing Guide](https://github.com/hixbe/font-files/blob/main/CONTRIBUTING.md).
